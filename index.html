Amazon Sales Analysis

This project aims analyze Amazon sales data using Python to identify key sales trends and insights. I started by cleaning the data, handling missing values and duplicates using pandas. After converting date fields and ensuring proper formatting, I performed exploratory data analysis to uncover trends, such as top-selling products and category-wise performance. I visualized monthly sales growth using matplotlib and created detailed bar charts for product analysis. Additionally, I applied time series analysis for sales forecasting and customer segmentation using clustering algorithms. Finally, I compiled all insights into visual reports for easy interpretation and integration with BI tool Power BI.

  • Technologies: Python, SQL, Jupyter Notebook
  • Code: Amazon Sales Analysis

Used Cars Analysis

This project is to analyze a used cars dataset, I began by importing essential libraries such as pandas, numpy, matplotlib, and seaborn for data manipulation and visualization. After downloading the dataset, I prepared it for analysis by cleaning and transforming the data, ensuring it was ready for exploratory data analysis (EDA). During the EDA phase, I examined relationships between features and the target variable, visualizing trends and distributions. For modeling and prediction, I implemented both Linear Regression and Random Forest algorithms, comparing their performance using metrics like mean squared error. Additionally, I conducted feature importance analysis for both models, utilizing SHAP values to interpret the contribution of each feature in the Random Forest model, which provided insights into the factors influencing used car prices.

  • Technologies: Deep Learning, Python
  • Code: Used Cars Analysis

Wine Quality Detection Analysis

The analysis explores the Portuguese "Vinho Verde" wine dataset, which includes physicochemical properties and quality ratings for red and white wines. Key findings indicate a strong correlation between alcohol content and wine quality, with most wines rated a quality of 6. The analysis also shows that while both wine types predominantly have quality ratings of 6, red wines tend to have higher quality ratings overall. Additionally, the heatmap illustrates varying correlations among the chemical characteristics, with some showing minimal impact on quality. A clean and consolidated dataset containing 6,497 records was prepared for further exploration.

  • Technologies: Python, Pandas,NumPy, Matplotlib, Seaborn, Jupyter Notebook
  • Code: Wine Quality Detection Analysis
